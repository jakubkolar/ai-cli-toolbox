# ai-cli-toolbox

CLI toolbox for AI agents - file summarization, PDF conversion, and utility scripts for Claude Code.

## Installation

```bash
uv tool install ai-cli-toolbox
```

## Development

```bash
just install    # Install dependencies
just check      # Run all checks (lint, typecheck, test)
```
